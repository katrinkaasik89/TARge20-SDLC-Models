Commands

to initialize a git repository to your project

to commit changes

- git branch

to see your branch

- git log

to see all your commits

- git checkout + id

- git checkout -- .

If you want revert all changes that you saved, but didn't add and didn't commit to git

- git checkout -b branch-name

- git branch

- git remote add origin "link-of-repository"

Checking connection

- git push -u origin master

to push branches to GitHub

- git fetch - git checkout origin/master - git merge origin/master

- git remote rm name-of-repository
